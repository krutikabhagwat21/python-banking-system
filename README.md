# python-banking-system
a console-based banking management 
 System developed using Python.

## Features

- Open New Account
- Login with PIN Authentication
- Transfer Money
- Monthly Interest Calculation
- Search Account by Name
- View All Accounts
- Bank Summary Dashboard

## Technologies Used

- Python
- Object-Oriented Programming (OOP)
- Functions & Modules
- Exception Handling

## Learning Outcomes

- Applied OOP concepts in a real-world project
- Implemented account management and transaction logic
- Improved problem-solving and Python development skills
